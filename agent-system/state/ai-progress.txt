## 使用说明

### 日志格式

每个会话应按以下格式记录：

```
## Session [日期时间]

### 目标
- 本次会话的目标

### 完成的工作
- 具体完成的任务

### 遇到的问题
- 问题描述和解决方案

### 下一步
- 下一个会话应该做什么

### 文件变更
- 新增/修改/删除的文件列表
```

### 状态标记

- ✅ 已完成
- 🔄 进行中
- ⏸️ 暂停
- ❌ 阻塞
- ⚠️ 需要注意
