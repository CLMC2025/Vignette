# 微语单词 V1 定稿设计文档

## 0. V1 目标与边界（上线铁律）

* 核心定位：**纯语境阅读记忆单词**（读语境 → 点不认识的词查释义 → 反馈熟练度 → 下一词）
* V1 必须做到：

  1. **开始学习稳定可用**（不因网络/AI 失败而中断）
  2. **离线可用**（至少能学习内置词书 + 离线语境兜底）
  3. **词书可用**（内置 + 导入 + 可选下载）
  4. **列表点词可看详情**（你强调的这个必须有）
* V1 暂不做（留到后续版本）：复杂社交、打卡社区、多题型训练、大量运营入口

---

## 1. 信息架构（页面结构）

V1 页面：

1. 首页（Home / Today）
2. 词书页（Wordbooks）
3. 单词列表页（Word List）
4. 学习页（Session）
5. 单词详情页（Detail / Sheet 或独立页）

## 2. 页面线框稿（文字版）

### 2.1 首页 Home / Today（唯一主入口）

**顶部栏**

* 左：应用图标 +「微语单词」
* 右：设置入口（齿轮）
  （通知中心、头像 V1 可不放，避免分散）

**中部主卡**

* 标语：通过微语境科学记忆单词
* 大按钮：`开始学习`（主色蓝，圆角 XL，高 56~64vp）
* 次按钮：`词书管理`（次要按钮）

**底部数据卡（轻量）**

* 今日进度：`已完成 X / 计划 Y`
* 今日到期复习：`R`
* 连续天数（可选显示，不做复杂动效也行）

**交互**

* 点「开始学习」：直接进入 Session（若未选词书 → 跳词书选择）
* 点「词书管理」：进入 Wordbooks

---

### 2.2 词书页 Wordbooks（系统词书 + 我的词书）

**结构：分组列表 + 每张词书卡**

* 分组 A：系统词书（内置）
* 分组 B：可下载词书（在线，可选下载）
* 分组 C：我的词书（用户导入/创建）

**词书卡信息**

* 名称：CET4 / CET6 / 考研 / 自定义…
* 词数：xxxx
* 状态：

  * 内置：可直接用
  * 可下载：显示“下载/已下载”
  * 我的词书：显示“导入来源/时间”
* 右侧主操作：

  * `选择学习`
  * 对“可下载词书”：`下载`（带进度条/百分比）

**底部操作**

* `导入词书`（单入口，进入 Import）

**交互**

* 点卡片：进入该词书的 Word List
* 点“选择学习”：设为当前词书（首页开始学习默认用它）

---

### 2.3 单词列表页 Word List

**顶部栏**

* 返回
* 词书名称（CET4…）
* 搜索框（可放在顶部下方）

**列表项（每行）**

* 左：英文单词
* 中：简短释义（若无则显示 “—”）
* 右：状态图标（新/学习中/复习/已掌握）

**交互**

* 点击任意单词 → 打开 Detail（建议“半屏弹层”，不打断）
* 长按（V1 可不做，避免复杂）
* 右上角：可选「开始学习」（使用本词书）

---

### 2.4 学习页 Session（纯语境阅读主场）

**顶部栏**

* 左：返回
* 中：`微语境学习`
* 右：撤回 Undo（仅上一个；无历史置灰）

**进度**

* 顶部次行：`第 X / Y 个`

**语境区（核心）**

* 一段英文语境（50~300字，按难度调整）
* **目标词高亮：下划线 + 轻背景色**
* **每个单词可点击**（你提的需求）：点击弹出 Detail 半屏卡（不跳页）

**释义区（默认折叠，但新手期可默认展开）**

* 提示行：`点击显示释义/翻译`
* 展开后：中文释义 + 关键义项（简洁）

**底部反馈按钮（固定三键）**

* `不认识`（红）
* `有点印象`（黄）
* `认识`（绿）

> 点击任一按钮：立即进入下一个词（按钮点击后 0.8~1s 防抖禁用，避免连点）

**忘了/不确定怎么办？**

* 直接点 `不认识` 即可，不需要二次确认
* 撤回用 Undo（放顶栏，不占底部按钮位）

---

### 2.5 单词详情 Detail（半屏弹层优先）

触发入口：

* Session 里点任意词
* Word List 点任意词

内容（V1 够用版）：

* 单词 + 音标（可选） + 发音按钮（TTS）
* 当前语境义（若有）
* 常用义项 1~3 条
* `加入单词本/收藏`（你说“只有不会才回去点”，完全符合：按钮放在详情里，不打扰主流程）
* 关闭：下滑/点空白区

---

### 2.6 设置页 Settings（克制但关键项齐）

**学习设置**

* 每日学习量（10~100）
* 自动发音（开关）
* 新手模式（开关，可选）

**AI 设置**

* API Base URL / Key / 模型
* 语境风格、难度等级

**离线与缓存**

* 离线模板开关（默认开）
* 语境缓存大小（建议：默认 10MB 或 1000 条；可调）

**同步（WebDAV）**

* 仅展示必要字段 + 2 个操作按钮（避免按钮泛滥）

  * 状态：上次同步时间/失败原因
  * 按钮：`立即同步`、`从云恢复`
* 密码/加密：同步密码（AES）
* 冲突策略：Last Modified Wins + Snapshot Backup（按你蓝图）

## 3. 词书“内置 + 可选下载”方案（V1 推荐做法）

### 3.1 合规建议（非常关键）

* **只使用明确开源许可** 的词表（例如 MIT/Apache/CC-BY 允许商用），并在应用内“词书信息/关于”页写清楚来源与许可证。
* 我搜到一个明确标注 **MIT License**、并包含 CET-4/CET-6/考研 等词表的开源仓库（可作为下载源/或整理后内置）：([GitHub][1])
* 另一个 CET 词表仓库也标注 MIT，但它说明来自考试大纲 PDF 的整理，商用前更建议优先选“来源更干净/更明确”的那类：([GitHub][2])

### 3.2 体积与体验：为什么要“可选下载”

* 安装包内置太多词表会变大、审核风险也更高
* V1 推荐：

  * **内置**：1 个“基础入门词书”（几百~1000词，保证离线可学）
  * **可选下载**：CET4 / CET6 / 考研（用户点下载才占空间）
  * **用户导入**：保留 Import（最稳妥、也最自由）

### 3.3 下载机制（产品侧表现）

在 Wordbooks 页的“可下载词书”分组里：

* 每本词书卡：显示大小、词数、许可证
* 按“下载”→ 进度条 → “已下载”
* 提供“删除已下载”释放空间（V1 必须有，不然容易越用越大）

---

## 4. 离线兜底与缓存


**语境生成优先级**

1. **缓存语境**（曾经 AI 生成过的，体积受控：上限/LRU）
2. **离线模板生成**（你已有离线模板体系）
3. **最小故事兜底**（保证包含目标词、长度短、语法正确）

缓存策略建议（V1 默认）：

* 上限：例如 **1000 条或 10MB**（二选一触顶）
* 淘汰：LRU（按 last_used_at）
* 设置页允许用户改上限/一键清空
